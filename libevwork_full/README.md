# libevwork


基于libev库封装，结合作者在一线互联网公司和手游行业多年的实战编程经验，编写的一套轻量级高性能网络引擎库。
主要特点：
1）支持protobuf、json两种主流通讯协议；
2）支持多线程，可用于开发IO阻塞性应用，也可用于开发多线程网络事件反应器应用（线程内创建事件循环）；
3）类MFC的编程风格；

主要应用：
1）目前已经用于几个知名的手游公司做服务器开发框架。
2）多个分布式网络应用，如kkeeper(paxos)、kescort。
